# CS 红楼 · 3D 弓箭博物馆

一个可在浏览器中运行的《红楼梦》第一人称 3D 博物馆。每个词条是一间展厅，展厅之间通过人物、地点、诗词、章回与事件关系相连，并加入了弓箭射击玩法。

## 操作

- `W A S D`：移动
- 鼠标：环顾与瞄准
- 左键：放箭
- `R`：搭箭
- `E`：进入准星指向的展厅门
- `F`：阅读当前展品
- `M`：打开关系图谱
- `/`：搜索展厅

## 本地运行

项目使用 ES Modules，需要通过本地 HTTP 服务访问：

```powershell
python -m http.server 8765
```

浏览器打开 <http://localhost:8765/>。

## 技术

- Three.js / WebGL
- 原生 HTML、CSS、JavaScript
- 本地静态知识库与图片资源
